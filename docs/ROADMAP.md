# Roadmap

This document outlines logical next steps if the platform grows.

## Short-Term

- Add CSV upload support with server-side parsing.
- Provide a dataset schema registry.
- Add pagination for large dataset lists.
- Export cleaned datasets as JSON/CSV.

## Medium-Term

- Add authentication middleware and role-based access controls.
- Persist datasets in a database (PostgreSQL or MongoDB).
- Add advanced cleaning operations (outlier detection, null imputation).
- Introduce a real charting library (D3, Plotly, or Vega).

## Long-Term

- Real-time collaborative editing of dataset rows.
- Audit trails for dataset changes.
- Hosted deployment with multi-tenant support.
- Integrations with cloud storage providers.
